# **Linux Notes**

## **Basic Commands**

> [!TIP] Inspect the CPU architecture
> ```shell
> dpkg --print-architecture
> ```
> ```arm64```
-------------------------------------------------------------------------------
> [!TIP] Inspect the kernel
> ```shell
> uname -r  
> ```
> ```6.1.43-rockchip-rk3588```
-------------------------------------------------------------------------------
> [!TIP] Inspect the OS
> ```shell
> cat /etc/os-release  
> ``` 
> ```PRETTY_NAME="Debian GNU/Linux 11 (bullseye)"```  
> ```NAME="Debian GNU/Linux"```  
> ```VERSION_ID="11"```  
> ```VERSION="11 (bullseye)"```  
> ```VERSION_CODENAME=bullseye```  
> ```ID=debian```
